Know how good your statement is!
Disclaimer:

This program was made using 901 classified statements available on the web and the machine learning algorithm is just a Naive Bayes applied to the most common feature in each text. This program analyses the text based only on the words present, and does not make any context analysis. You can certainly ask the software to rate the sentence "beautiful flying cow" and it will give you a score and an error, though you can obviously imagine this result has no meaning whatsoever.

However, it is in fact quite good at rating entire statements, classifying the high majority of my training set correctly (it also rated my own statement very highly, proving that the algorithm is obviously working!).
